# Probabilistic-Models-for-FFP-Interpretation
This study quantifies uncertainties in interpreting free-fall penetrometer (FFP) data in clay and develops predictive models for static tip resistance using FFP acceleration and velocity via Bayesian inference and three probabilistic MLP methods: MaxLike, MaxLike + MCD, and Bayesian MLP-VI.
